# Customer Behavior Analysis

## Overview
This project analyzes customer purchasing behavior using Power BI, SQL, and Python.

## Features
- Customer segmentation
- Revenue analysis
- Purchase trends
- Interactive dashboard

## Tools Used
- Power BI
- SQL
- Python
